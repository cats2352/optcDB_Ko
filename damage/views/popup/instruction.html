사용법 안내
원하는 동작을 누른 상태에서 클릭하여 작동 방식을 변경할 수 있습니다.

데미지 창: (오른쪽)

  • 마우스 가운데 버튼 클릭: 미스 → 빗나감 → 굿 → 그레이트 → 퍼펙트 순으로 변경
  • CTRL + 클릭: 미스 → 빗나감 → 굿 → 그레이트 → 퍼펙트 순으로 변경
  • SHIFT + 클릭: 적의 속성을 변경 (단일 대상)
  • Drag & Drop: 공격 순서 변경
  • 배경 더블 클릭: 다크 모드 켜기/끄기

선장/선원 선택 창: (중앙 영역)
  • 마우스 가운데 버튼 클릭: 미스 → 빗나감 → 굿 → 그레이트 → 퍼펙트 순으로 변경
  • CTRL + 클릭: 슬롯 일치 → 반대 → 없음 (슬롯을 클릭해도 변경 가능)
  • SHIFT + 클릭: 선장효과 무효 상태로 만듬
  • ALT + 클릭: 선장/선원 봉쇄
  • SHIFT + ALT + 클릭: 선장 및 선원 비활성화
  • Drag & Drop: 선장/선원 위치 변경
  • 레벨 클릭: 레벨설정창 열기
  • 레벨 마우스 가운데 버튼 클릭: 선장/선원을 최대 레벨로 설정

선박 메뉴설정: (설정 메뉴 하단)
  • CTRL + 클릭: 선박 선택 창 열기

적 방어력: (설정 메뉴 하단)
  • CTRL + 클릭: 방어력 선택창 열기

스텟 창: (선장/선원 창 아래)
  • 슬라이더 드래그: 체력 비율 조절
  • 마우스 왼쪽 버튼 클릭 or 체력 클릭: HP 수치 또는 % 직접 입력
